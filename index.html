Air Piano

It's a simulation of a piano in the air (like air guitar) using Arduino and Processing.
It is done through the detection of hands that should be between 70 and 200 centimeters and measure distances between the Sensor SR04 emitting a sound of each key "typed in the air".

Source Code

Submarine Sonar

It's a simulation of a submarine sonar using Arduino and Processing.
It is done through the detection of objects and measuring distances between the Sensor SR04 and the object that approaches it, emitting a sonar sound and another of an alarm each time the distance is "dangerous"
