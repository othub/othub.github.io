Telemedicine API

Decentralized API that follows the Solid Specifications .
Using the proxy design pattern, this api allows the original telemedicine Spring application developed by Izertis to store data in a decentralized database that, in this case, will be the a decentralized POD in localhost

Air Piano

It's a simulation of a piano in the air (similar to air guitar) using Arduino and Processing.
It is done through the detection of hands that should be between 70 and 200 centimeters and measure distances between the Sensor SR04 emitting a sound of each key "typed in the air".
